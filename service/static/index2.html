Promotion Demo REST API Service

GET /promotions to see all the promotions.

GET /promotions?category=BOGO to see all of the promotions in the 'BOGO' category.

GET /promotions/{id} retrieves a promotion from the DB using an ID.

PUT /promotions/{id} updates a promotion using an ID.

DELETE /promotions/{id} removes a promotion from the DB using an ID.

POST /promotions creates an new promotion.

Example Data:

    {
        "productid": "A123",
        "category": "BOGO",
        "available": "Y",
        "discount": "10"
    }
